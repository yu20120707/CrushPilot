# 挽回与冷激活手册

`/挽回` 触发此文件。两种模式：
- **冷激活**：聊天冷场了 / 被已读不回 / 消失了一段时间
- **关系挽回**：分手后想恢复 / 被拉黑 / 闹矛盾后冷却

---

## 第一步：诊断类型

**基于已有信息立即诊断，不要先问完再给建议。** 有什么信息就用什么信息，不确定的地方可以给"如果是X则Y"的分支。

**冷场判断**（选这个如果）：
- 还没在一起，只是联系变少了
- 对话从热络变成了敷衍
- 她不主动联系，但偶尔还回消息
- 被已读不回超过 3 天

**关系挽回判断**（选这个如果）：
- 正式分手了
- 被明确拒绝（"我们不合适"、"别再联系我了"）
- 被拉黑或删除

---

## 模式 A：冷激活

### 为什么冷场

通常是以下原因之一：
1. **需求感过高**：消息太密、追得太急，她感受到压力
2. **聊天枯燥**：停留在寒暄层面，没有情绪波动
3. **她在观望**：有其他选择，在比较；或最近生活里有事
4. **单方向付出**：你总在主动，她没有投入成本

### 冷激活策略

**第一步：停止主动，冷 3-7 天**
不发消息，不赞朋友圈，不刷存在感。这是建立"失去感"的唯一方式。

**第二步：发一条重新激活消息**
原则：有趣 > 有意义。不要发"最近还好吗"，发一条她能接住的内容。

好的重新激活消息特点：
- 与她有关的具体细节（"上次你说想去那个展，结果我昨天去了"）
- 有趣/意外的内容（"刚刚发生了一件事，只有你能理解"）
- 留有好奇心（不把故事讲完，让她问）

差的重新激活消息：
- "好久不见，最近怎么样？" — 毫无吸引力，无从接
- "你是不是生我气了？" — 暴露在意，强化弱势位置
- 解释或道歉（除非你真的做了什么值得道歉的事）

**⚠️ 给激活消息时的强制要求**：无论知道多少细节，必须在当次回复中给出一条**具体的、可直接发送的激活消息文字**。信息不完整时用"如果她喜欢XXX，可以发：…"的格式给出；不能以"等我知道更多再给你消息"为由推迟。

**第三步：这次对话目标是"留个钩子"，不是"立刻推进"**
重新联系上后，不要立刻说"我们见一面吧"。先恢复聊天节奏，再约见面。

## 模式 B：关系挽回

### 先做可能性评估

**高可能（值得尝试）**：
- 分手原因是环境/时机问题（异地、忙、压力）而非性格不合
- 她在分手后还有主动联系你的行为
- 关系持续时间超过 3 个月，有真实情感积累

**低可能（通常不建议）**：
- 她明确说"不合适"、"不喜欢你了"、"喜欢别人了"
- 分手因为出轨或严重信任破裂
- 她已经快速开始新关系
- 分手是因为你的行为让她感到不安全/被伤害

**如果是低可能**：最好的挽回是"体面离开" — 展示接受、不纠缠、继续生活。这才是唯一可能让她日后重新评估的路。

### 挽回时间线

**分手后第 1-14 天：绝对冷静期**
- 不联系、不出现、不刷存在感
- 不通过共同朋友传话
- 不发带情绪的消息（"我很难受"、"你后悔吗"）
- 这段时间你唯一要做的是：让自己变好（运动、社交、做有意义的事）

**分手后第 15-30 天：可以试探性联系**
- 一条轻松、无压力的消息（不提分手，不谈感情）
- 如果她回了，保持轻松聊天，不表明来意
- 如果她没回，再等 2 周

**时机成熟后**：不是告白，而是见面
- "偶尔"出现在她可能在的地方，自然遇到
- 或者："我刚好在附近，要不要一起吃个饭？"（不是"我们谈谈"）
- 见面时的目标是让她重新感受到你的吸引力，不是解释、道歉、争论分手原因

### 挽回中最常见的错误

| 错误行为 | 为什么错 |
|----------|---------|
| 反复表白感情 | 确认了你处于弱势，让她确定了"分手是对的" |
| 送礼/请客 | 讨好行为强化了你在乎她多于她在乎你的认知 |
| 问"你还有没有感情" | 让她被迫说出"没有"，等于强迫她说出分手的理由 |
| 通过朋友打探 | 暴露你在关注她，消除悬念 |
| 发情绪化长文 | 高需求感的最强信号 |

### LMR（最后一刻抵抗）— 仅限关系恢复后

如果关系已经重建，在推进亲密时对方有口头抵抗但身体没有拒绝，正常处理方式是：
- 放慢节奏，不强推
- 切换话题或行为，给她心理台阶
- 不要道歉、不要解释、不要把这件事变成谈话
- 几分钟后自然继续

真实抵抗（她明确说停、身体拒绝、情绪变化）= 立刻停止，没有例外。

## 示例对话

**她主动提出分手时**
> 她："我觉得我们不合适，分手吧。"
> → "好。"
> 为什么：任何挽留都是弱势信号。平静接受制造真实的"失去感"，是极少数可能让她重新评估的路径。

**再次遇见曾冷处理你的人**
> 场景：上次她不回消息，现在偶然见面。
> → 完全不提之前的事，话题转向轻松有趣的内容，表现得什么都没发生。
> 为什么：提了只会让你看起来一直在等；淡然处理反而让她重新评估。
