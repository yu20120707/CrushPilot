# 展示面诊断指南

`/展示面` 触发此文件。用户发截图时直接视觉分析；没截图时先问平台和基本情况。

---

## 诊断流程

**第一步**：确认平台（微信/探探/Tinder/Soul/陌陌/Bumble/青藤之恋）
**第二步**：收集材料（头像、动态/朋友圈截图、个人简介）
**第三步**：按平台维度打分（见下）
**第四步**：输出评分 + 3个最大问题 + 具体改进方案 + 改进前后对比示例

输出格式：
```
平台：xxx  总分：x/10

各维度：
- 头像：x/10 — [诊断]
- 简介/Bio：x/10 — [诊断]
- 内容/动态：x/10 — [诊断]

三大问题：
1. ...
2. ...
3. ...

具体改法：
1. [可直接执行的行动]
2. ...

朋友圈/动态内容比例建议（微信/陌陌必须输出）：
生活 40% + 兴趣爱好 30% + 社交 20% + 工作 10%

改进示例：
改前：[原内容]
改后：[建议版本]
```

---

## 微信

**展示面 = 头像 + 个性签名 + 朋友圈**

**头像评分维度（/10）**
- 清晰度：能看清脸，非背影/侧脸/戴口罩/风景代替
- 状态感：自然不做作，有活力或有故事感（旅行、运动、聚会）
- 印象分：第一眼想不想点进去看

**朋友圈内容比例（最近30条）**
理想比例：生活40% + 兴趣爱好30% + 社交20% + 工作10%
红线：纯鸡汤/广告/转发 > 50%，说明没有真实生活；全是工作/打卡 > 60%，显得无聊。

**朋友圈内容质量**
- 加分：有故事的照片（"昨晚做了一道失败的烤鸭"比"美食打卡"好）
- 加分：偶尔展示社交活动（你在聚会，不是在家刷手机）
- 加分：有自己观点的评论/转发（体现有独立思想）
- 减分：自拍超过60%（显得自恋）
- 减分：频繁发无聊的日常流水账

**个性签名**
- 好：留有好奇心的一句话（"在想一个问题"、"最近在学攀岩，比我以为的难"）
- 差：鸡汤句子、空着、工作头衔

## 探探 / Tinder

**展示面 = 照片组合 + 简介**

**照片策略（6张）**
- 第1张（封面）：清晰正脸，自然微笑，环境有故事感；绝对不用合照/背影/戴墨镜
- 第2-3张：展示生活方式（运动/旅行/爱好的行动照，不是摆拍）
- 第4张：社交照（和朋友/聚会，证明你不是宅男）
- 第5-6张：可以放正脸特写 + 一张能展示身材/体型的全身照

**绝对禁止**：镜子自拍、侧脸、车里自拍、健身房镜子、抱孩子照

**简介（探探/Tinder）**
- 字数：50-120字，太短无聊，太长没人读
- 公式：一个具体细节的生活描述 + 一个轻幽默 + 一个开放式邀约
- 示例：`爬山路上会找路的那种人 | 做饭一般但摆盘很好看 | 你愿意尝尝我的"美食"吗`
- 禁止：罗列标签（"爱旅游爱美食爱运动"=等于没说）

## Soul

**展示面 = 文字 + 声音 + 人设一致性**

Soul 的核心逻辑：深度 > 外表，文字质量决定匹配质量。

**自我介绍**：要有一个有趣/反常的细节，不能是"喜欢音乐电影美食"
**语音片段**：音质清晰，语速适中，内容有点故事感（不要直接读自我介绍）
**发帖内容**：观点型 > 日记型。"我发现有个规律：……你们有同感吗？"比"今天天气很好"好

---

## 陌陌

**展示面 = 头像 + 动态 + 附近展示**

和微信类似但更直接。附近功能重要——多发带位置感的动态，证明你是真实活跃的本地人。

---

## Bumble

**展示面 = 照片 + Prompt回答（让她主动发消息的关键）**

Bumble 的特殊性：女生第一条消息决定一切。你的任务是给她一个"忍不住回复"的钩子。

**Prompt 回答公式**：用具体、有点反常的内容制造好奇心
- 差：`我的爱好是旅行和健身` 
- 好：`我在成都学了三个月做泡菜，现在完全停不下来`

**最后一张照片**：加上 prompt "如果你不知道说什么就从这里开始↑"配上一个有争议性的细节照片（书架/奇怪的收藏品/有故事的地方）

---

## 青藤之恋

**展示面 = 学历认证 + 深度内容**

这个平台用户期待真诚和深度。

- 认证学历：强烈建议，是这个平台的信任锚
- 自我介绍：要比其他平台更有深度，可以分享一个人生观或正在思考的问题
- 不要过度包装：这个平台的用户对表演型简介敏感

---

## 通用红线（任何平台）

1. 头像是多年前照片 → 真实见面落差感，直接损害信任
2. 简介全是标签 → 没有记忆点
3. 社交媒体完全空着/锁着 → 显得有秘密或无聊
4. 所有照片都在家/室内 → 生活圈子太窄的信号
5. 配图是车/名表/豪华酒店（没有你自己）→ 显得不安全感强
