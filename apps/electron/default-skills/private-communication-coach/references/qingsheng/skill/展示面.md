name: 展示面
description: "情圣子命令：展示面诊断。分析你在微信、探探、Soul、Bumble等平台上的头像、朋友圈、个人简介，给出评分和改进方案。直接发截图就能分析。"
---

读取情圣主技能文件 SKILL.md 和 `references/profile-audit.md`，然后进入展示面诊断模式：

**有截图**：用户直接发头像/朋友圈/个人主页截图，直接视觉分析，不需要文字描述。

**没截图**：先问平台和基本情况（3句话搞定）。

输出格式：
1. 总分（1-10）+ 各维度打分
2. 最大的 3 个问题（直说，不绕弯）
3. 具体改法（可以立刻执行的）
4. 改前 vs 改后对比示例
